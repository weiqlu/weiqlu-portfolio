## Tech Stack

- **Frontend:** React + Vite
- **Styling:** Material UI (MUI), Framer Motion
- **Infrastructure:** AWS S3, CloudFront
- **CI/CD:** GitHub Actions

## Local Development 

```bash
cd client
npm install
npm run dev
```

## Build

```bash
cd client
npm run build
npm run preview
```

## Deployment

Deployed to **AWS S3 + CloudFront** via **GitHub Actions** on pushes to `main`.
